# SiriFlow-To-Do-List

This is my first web development project! It is a To-Do List web application built using Python and the Flask framework.

## Features
- **Task Management:** Users can add, view, and manage their daily tasks.
- **User Authentication:** Includes secure Login and Signup functionality.
- **Responsive UI:** Built with HTML, CSS, and JavaScript for a clean user experience.

## Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Version Control:** Git & GitHub
- **Deployment:** Render
